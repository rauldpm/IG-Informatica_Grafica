//*******************************************************************//

		INSTRUCCIONES PARA COMPILAR Y EJECUTAR

1 - En la terminal, estando en la carpeta con todos los archivos,
escribir "make" para compilar el programa (SOLO UNA VEZ SI NO SE BORRAN ARCHIVOS).

2 - Para ejecutar, escribir: "./bin/practica4"


//*******************************************************************//

			INSTRUCCIONES DE USO

1 - Introduce "S" si se quiere cargar un archivo/imagen/datos por teclado,
 							"N" si no se quiere cargar nada.

	1.1 - Si se elige no cargar ningun archivo desde un ply -> opcion "N"

			(Activar con tecla "p")
			Tecla: "1" -> selecciona el dibujado de un octaedro.
			Tecla: "2" -> selecciona el dibujado de un tetraedro.
			Tecla: "3" -> selecciona el dibujado de un cubo.
			Tecla: "4" -> NO HAY OBJETO POR CARGA, NO SE DIBUJA NADA.
			Tecla: "5" -> selecciona el dibujado de revolucion de la prueba 2.
			Tecla: "6" -> selecciona el dibujado de un objeto jerarquico (grua).
			Tecla: "7" -> selecciona el dibujado de un objeto textura

	1.2 - Si se elige introducir por teclado los vertices a revolucionar

			(Activar con tecla "p")
			Tecla: "1" -> selecciona el dibujado de un octaedro.
			Tecla: "2" -> selecciona el dibujado de un tetraedro.
			Tecla: "3" -> selecciona el dibujado de un cubo.
			Tecla: "4" -> NO HAY OBJETO POR CARGA, NO SE DIBUJA NADA.
			Tecla: "5" -> selecciona el dibujado de revolucion de los valores introducidos por teclado.
			Tecla: "6" -> selecciona el dibujado de un objeto jerarquico (grua).
			Tecla: "7" -> selecciona el dibujado de un objeto textura

	1.3 - Si se elige cargar un archivo desde un ply como carga -> opcion "S" y "C"

			(Activar con tecla "p")
			Tecla: "1" -> selecciona el dibujado de un octaedro.
			Tecla: "2" -> selecciona el dibujado de un tetraedro.
			Tecla: "3" -> selecciona el dibujado de un cubo.
			Tecla: "4" -> selecciona el dibujado de un archivo por carga.
			Tecla: "5" -> NO HAY OBJETO POR CARGA, NO SE DIBUJA NADA.
			Tecla: "6" -> selecciona el dibujado de un objeto jerarquico (grua).
			Tecla: "7" -> selecciona el dibujado de un objeto textura

	1.4 - Si se elige cargar un archivo desde un ply como revolucion -> opcion "S" y "R"

			(Activar con tecla "p")
			Tecla: "1" -> selecciona el dibujado de un octaedro.
			Tecla: "2" -> selecciona el dibujado de un tetraedro.
			Tecla: "3" -> selecciona el dibujado de un cubo.
			Tecla: "4" -> NO HAY OBJETO POR CARGA, NO SE DIBUJA NADA.
			Tecla: "5" ->	selecciona el dibujado de un archivo por revolucion.
			Tecla: "6" -> selecciona el dibujado de un objeto jerarquico (grua).
			Tecla: "7" -> selecciona el dibujado de un objeto textura

2 - Pulsa las siguientes teclas en el teclado para alternar
el modo de visualizado:

	Tecla: "m" -> Activa menu movimiento luces
		Tecla: "1" -> Disminuye x de luz 1 y/o 2
		Tecla: "2" -> Disminuye y de luz 1 y/o 2
		Tecla: "3" -> Disminuye z de luz 1 y/o 2
		Tecla: "4" -> Aumenta x de luz 1 y/o 2
		Tecla: "5" -> Aumenta y de luz 1 y/o 2
		Tecla: "6" -> Aumenta z de luz 1 y/o 2
		Tecla: "7" -> Resetea a 0 la x de luz 1 y/o 2
		Tecla: "8" -> Resetea a 0 la y de luz 1 y/o 2
		Tecla: "9" -> Resetea a 0 la z de luz 1 y/o 2

	Tecla: "p" -> Activa menu figuras
		Tecla: "1" -> Octaedro
		Tecla: "2" -> Tetraedro
		Tecla: "3" -> Cubo
		Tecla: "4" -> Ply carga
		Tecla: "5" -> Ply revolucion
		Tecla: "6" -> Jerarquia
		Tecla: "7" -> Textura

	Tecla: "o" -> Activa menu dibujos
		Tecla: "1" -> Puntos
		Tecla: "2" -> Lineas
		Tecla: "3" -> Relleno
		Tecla: "4" -> Ajedrez
		Tecla: "5" -> Prueba1 (mixto)

	Tecla: "i" -> alterna general luces
	Tecla: "u" -> alterna sombreado
	Tecla: "y" -> alterna tapa de revolucion

	Si el modo de figura es 6 (jerarquico)
		Tecla: "-" -> dibuja el revolucionado con menos revoluciones.
		Tecla: "+" -> dibuja el revolucionado con mas revoluciones.
		Tecla: "Z" -> aumenta el primer grado de libertad (giro).
		Tecla: "z" -> disminuye el primer grado de libertad (giro).
		Tecla: "X" -> aumenta el segundo grado de libertad (brazo extensible).
		Tecla: "x" -> disminuye el segundo grado de libertad (brazo extensible).
		Tecla: "C" -> aumenta el tercer grado de libertad (cuerda).
		Tecla: "c" -> disminuye el tercer grado de libertad (cuerda).
		Tecla: "V" -> aumenta el cuarto grado de libertad (prueba3).
		Tecla: "v" -> disminuye el cuarto grado de libertad (prueba3).
		Tecla: "A" -> Aumenta la velocidad del primer grado de libertad.
		Tecla: "a" ->	Disminuye la velocidad del primer grado de libertad.
		Tecla: "S" -> Aumenta la velocidad del segundo grado de libertad.
		Tecla: "s" -> Disminuye la velocidad del segundo grado de libertad.
		Tecla: "D" -> Aumenta la velocidad del tercer grado de libertad.
		Tecla: "d" -> Disminuye la velocidad del tercer grado de libertad.
		Tecla: "F" -> Aumenta la velocidad del cuarto grado de libertad (prueba3).
		Tecla: "f" -> Disminuye la velocidad del cuarto grado de libertad (prueba3).
		Tecla: "q" -> Pone a 0 la velocidad del primer grado de libertad.
		Tecla: "w" -> Pone a 0 la velocidad del segundo grado de libertad.
		Tecla: "e" -> Pone a 0 la velocidad del tercer grado de libertad.
		Tecla: "R" -> Activa el modo animacion (grua).
		Tecla: "r" -> Desactiva el modo animacion (grua).

	Si el modo de figura es el 5 (revolucion)
		Tecla: "T" -> Activa Tapas
		Tecla: "t" -> Desactiva Tapas

	Teclas generales
		Tecla: "/" -> reduce el zoom.
		Tecla: "*" -> aumenta el zoom.

	Teclas especiales
		Tecla: "F1" -> Enciende/Apaga luz 1
		Tecla: "F2" -> Enciende/Apaga luz 2
		Tecla: "F3" -> Mueve luz 1 (rotacion)
		Tecla: "F4" -> Mueve luz 2 (rotacion)
		Tecla: "F5" -> Activa material 1
		Tecla: "F6" -> Activa material 2
		Tecla: "F7" -> Activa material 3

3 - Pulsa "ESC" en el teclado para salir del programa.

//*******************************************************************//

		INSTRUCCIONES DE LIMPIEZA Y EMPAQUETADO

1 - En la terminal, estando en la carpeta con todos los archivos,
escribir "make clean" para borrar todos los archivos objeto y ejecutables.

2 - En la terminal, estando en la carpeta con todos los archivos,
escribir "make redo" para rehacer todos los archivos .cc y compilar el programa.

3 - En la terminal, estando en la carpeta con todos los archivos,
escribir "make superclean" para borrar todos los archivos objeto, ejecutables
y archivos temporales.
