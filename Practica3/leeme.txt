
Practica 3 -- Modelos Jerarquicos

//*******************************************************************//

Este programa dibuja un Tetraedro, un Cubo, un Octaedro, un archivo PLY
en modo carga o un archivo PLY en modo revolucion dada un perfil de dibujado
y una Grua.

El programa se inicia con la visualizacion de Tetraedro en modo Puntos.

Usa las flechas de direccion del teclado para girar la figura.

Si hubiera que cambiar o introducir un ply por teclado o en el codigo,
hay que indicar la ruta "data/(nombre).ply"

//*******************************************************************//

		INSTRUCCIONES PARA COMPILAR Y EJECUTAR

1 - En la terminal, estando en la carpeta con todos los archivos,
escribir "make" para compilar el programa (SOLO UNA VEZ SI NO SE BORRAN ARCHIVOS).

2 - Para ejecutar, escribir: "./bin/practica3"


//*******************************************************************//

			INSTRUCCIONES DE USO

1 - Introduce "S" si se quiere cargar un archivo, "N" si no se quiere cargar un archivo
o introduce "T" si se quiere introducir los valores por teclado para revolucionar:

	1.1 - Si se elige no cargar ningun archivo desde un ply -> opcion "N"

			Tecla: "0" -> selecciona el dibujado de un octaedro.
			Tecla: "1" -> selecciona el dibujado de un tetraedro.
			Tecla: "2" -> selecciona el dibujado de un cubo.
			Tecla: "3" -> NO HAY OBJETO POR CARGA, NO SE DIBUJA NADA.
			Tecla: "4" -> selecciona el dibujado de revolucion de la prueba 2.
			Tecla: "5" -> selecciona el dibujado de un objeto jerarquico (grua).

	1.2 - Si se elige introducir por teclado los vertices a revolucionar -> opcion "T"

			Tecla: "0" -> selecciona el dibujado de un octaedro.
			Tecla: "1" -> selecciona el dibujado de un tetraedro.
			Tecla: "2" -> selecciona el dibujado de un cubo.
			Tecla: "3" -> NO HAY OBJETO POR CARGA, NO SE DIBUJA NADA.
			Tecla: "4" -> selecciona el dibujado de revolucion de los valores introducidos por teclado.
			Tecla: "5" -> selecciona el dibujado de un objeto jerarquico (grua).

	1.3 - Si se elige cargar un archivo desde un ply como carga -> opcion "S"

			Tecla: "0" -> selecciona el dibujado de un octaedro.
			Tecla: "1" -> selecciona el dibujado de un tetraedro.
			Tecla: "2" -> selecciona el dibujado de un cubo.
			Tecla: "3" -> selecciona el dibujado de un archivo por carga.
			Tecla: "4" -> NO HAY OBJETO POR CARGA, NO SE DIBUJA NADA.
			Tecla: "5" -> selecciona el dibujado de un objeto jerarquico (grua).

	1.4 - Si se elige cargar un archivo desde un ply como revolucion -> opcion "S"

			Tecla: "0" -> selecciona el dibujado de un octaedro.
			Tecla: "1" -> selecciona el dibujado de un tetraedro.
			Tecla: "2" -> selecciona el dibujado de un cubo.
			Tecla: "3" -> NO HAY OBJETO POR CARGA, NO SE DIBUJA NADA.
			Tecla: "4" ->	selecciona el dibujado de un archivo por revolucion.
			Tecla: "5" -> selecciona el dibujado de un objeto jerarquico (grua).

2 - Pulsa las siguientes teclas en el teclado para alternar
el modo de visualizado:

	Tecla: "p" -> dibuja la figura en modo Puntos.
	Tecla: "l" -> dibuja la figura en modo Aristas/Alambres.
	Tecla: "f" -> dibuja la figura en modo Relleno.
	Tecla: "v" -> dibuja la figura en modo Ajedrez.
	Tecla: "h" -> dibuja la figura en modo Puntos, Aristas y Relleno.
	Tecla: "T" -> dibuja el revolucionado con tapas.
	Tecla: "t" -> dibuja el revolucionado sin tapas.
	Tecla: "-" -> dibuja el revolucionado con menos revoluciones.
	Tecla: "+" -> dibuja el revolucionado con mas revoluciones.
	Tecla: "/" -> reduce el zoom.
	Tecla: "*" -> aumenta el zoom.
	Tecla: "Z" -> aumenta el primer grado de libertad (giro).
	Tecla: "z" -> disminuye el primer grado de libertad (giro).
	Tecla: "X" -> aumenta el segundo grado de libertad (brazo extensible).
	Tecla: "x" -> disminuye el segundo grado de libertad (brazo extensible).
	Tecla: "C" -> aumenta el tercer grado de libertad (cuerda).
	Tecla: "c" -> disminuye el tercer grado de libertad (cuerda).
	Tecla: "A" -> Activa el modo animacion (grua).
	Tecla: "a" -> Desactiva el modo animacion (grua).
	Tecla: "B" -> Aumenta la velocidad del primer grado de libertad.
	Tecla: "b" ->	Disminuye la velocidad del primer grado de libertad.
	Tecla: "N" -> Aumenta la velocidad del segundo grado de libertad.
	Tecla: "n" -> Disminuye la velocidad del segundo grado de libertad.
	Tecla: "M" -> Aumenta la velocidad del tercer grado de libertad.
	Tecla: "m" -> Disminuye la velocidad del tercer gradod e libertad.
	Tecla: "q" -> Pone a 0 la velocidad del primer grado de libertad.
	Tecla: "w" -> Pone a 0 la velocidad del segundo grado de libertad.
	Tecla: "e" -> Pone a 0 la velocidad del tercer grado de libertad.

3 - Pulsa "ESC" en el teclado para salir del programa.

//*******************************************************************//

		INSTRUCCIONES DE LIMPIEZA Y EMPAQUETADO

1 - En la terminal, estando en la carpeta con todos los archivos,
escribir "make clean" para borrar todos los archivos objeto y ejecutables.

2 - En la terminal, estando en la carpeta con todos los archivos,
escribir "make redo" para rehacer todos los archivos .cc y compilar el programa.

3 - En la terminal, estando en la carpeta con todos los archivos,
escribir "make superclean" para borrar todos los archivos objeto, ejecutables
y archivos temporales.
